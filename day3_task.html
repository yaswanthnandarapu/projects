Visual Studio Code

What is Visual Studio Code?

Visual Studio Code (VS Code) is a free, open-source code editor developed by Microsoft. It’s lightweight but powerful, providing features that make it suitable for both beginners and experienced developers. It helps to work on many programming languages in a single application or platform.

Key Features:

  1. Code Editing
  2. Extensions
  3. Version Control integration(Git)
  4. Debugging
  5. Cross-platform
  6. Customizability
  7. Integrated Terminal
  8. Lightweight and Fast
  9. Remote Devlopment

Emmet Live:

Emmet Live in Visual Studio Code (VS Code) refers to the ability to use Emmet for real-time coding enhancements, particularly when working with HTML and CSS. Emmet is a built-in feature in VS Code that allows you to write code faster by expanding abbreviations into full-fledged HTML or CSS structures.

  1. Abbreviation Expansion: Quickly write tags like div.container or ul>li*3 and expand them into full HTML structures.
  2. Live Updates: Combine Emmet with extensions like Live Server or Live Preview to see changes immediately as you code.
  3. CSS Shortcuts: Write CSS properties using abbreviations, such as m10 for margin: 10px;.
  4. Customizable: Configure Emmet settings to match your coding style and needs.
Abbreviation Uses
! Creates the basic structure of the html code.
> Creates a child element. For example, ul > li creates a ul element with a li child element.
^ used to navigate up one level in the HTML structure and add a sibling element. It’s especially useful when you need to move out of nested structures and continue building elements.
+ Creates a sibling element. For example, div + p will create a div and a p next to each other.
* Repeats the element a specified number of times. For example, ul > li*5 creates a ul with 5 li items inside.
{} Used to insert text or content inside an element. For example, div{Hello} creates a containing the text "Hello."
[] used in abbreviations to add attributes to HTML elements. This allows you to include custom attributes like id, class, data-*, or any other valid HTML attribute within your shorthand.
