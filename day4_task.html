Frontend Technologies:

Frontend technologies are used to build the user-facing part of web applications. They include languages, frameworks, and libraries that help in designing and building interactive, responsive, and visually appealing websites.

  • HTML
  • CSS
  • JavaScript
  • React
  • Angular
  • Vue.js
  • Bootstrap
  • SASS
  • jQuery
  • TypeScript

Backend Technologies:

Backend technologies handle the server-side logic, database interaction, and application processing that users do not see. These technologies ensure that the application performs its tasks and handles data appropriately.

  • Java
  • Node.js
  • Express.js
  • MongoDB
  • MySQL

MERN:

MERN Stack is a JavaScript Stack that is used for easier and faster deployment of full-stack web applications.

MERN Stack comprises of 4 technologies namely:
MongoDB
Express.js
React
Node.js

MEAN:

MEAN Stack is one of the most popular Technology Stack. It is used to develop a Full Stack Web Application. Although it is a Stack of different technologies, all of these are based on JavaScript language.

MEAN Stack comprises of 4 technologies namely:
MongoDB
Express.js
Angular
Node.js
